# ToastyClassChores
A World of Warcraft addon designed to help keep track of important class upkeep things such as shadowform and hunter pets. 

Currently Implemented:
- Shadowform
- Raid buffs (Out of combat)
- Hunter pets
- Warlock pets
- DK pets
- Druid forms
- Warrior stances
- Paladin auras
- Grimoire of Sacrifice reminder

Planned Features:
- Rogue poisons
- Raid buffs (Solo)

Am looking into:
- Source of Magic
- Earth Shield
- In combat rebuff reminder
