# ToastyClassChores
A World of Warcraft addon designed to help keep track of important class upkeep things such as shadowform and hunter pets. 

Currently Implemented:
- Shadowform
- Raid buffs (Out of combat)
- Hunter pets
- Warlock pets
- DK pets
- Druid forms
- Warrior stances
- Paladin auras
- Rogue poisons
- Grimoire of Sacrifice reminder
- Raid buffs (Solo)

Planned Features:
- Lightning/Water Shield

Am looking into:
- Source of Magic
- Earth Shield
- In combat rebuff reminder
- Early warning for rogue poisons and raid buffs falling off
