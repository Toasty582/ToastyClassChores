# ToastyClassChores
A World of Warcraft addon designed to help keep track of important class upkeep things such as shadowform and hunter pets. 

Currently Implemented:
- Shadowform
- Raid buffs (Out of combat)
- Hunter pets
- Warlock pets
- Druid forms

Planned Features:
- Raid buffs (Solo)
- Warrior stances
- Grimoire of Sacrifice reminder

Want to do but don't know how yet:
- Raid buffs (In combat)
- Source of Magic
- Earth Shield
- DK pets
- Rogue poisons
- Paladin auras
